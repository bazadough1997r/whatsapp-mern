# WhatsApp Clone 
This clone is just for practicing how to make MongoDB real time.

# Technologies
MERN Stack (MongoDB, Express.js, React.js, Node.js)
- Node Express
- Reac js
- MongoDB Atlas
- React-pusher
- Pusher-js
- Material UI

### Installing Dependencies
Open 2 terminal windows:
```sh
cd backend
npm install
npm start

cd frontend
npm install
npm start
```
